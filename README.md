# AvaTown-VR-Avatar-Site

## Description
**AvaTown**, is a online marketplace that brings together consumers looking for VR Avatars.

## Tech
### Front-end
Reactjs

## Run the Project
To run this projects, install the following packages and dependencies,

```
npm install -g sass
npm i react-router-dom
npm run dev
```
